Coffee card

The uniform card used on the Espresso page (list view and map modal). One skeleton per card: header (name + roaster), a details block of labelled rows, then note slots. Any section without content — and every spare note slot — carries the “hidden” attribute and collapses, so a card only shows the rows it has. The favorite banner is a binary flag (data-fav) whose year comes from the card’s data-year. Toggle the sections below to see it adapt.
